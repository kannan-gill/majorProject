ABOUT

Dip Coater Unit aims to make a production ready dip coating unit with various user given parameters in order to influence the thickness of coating on the substrate.
These user-given parameters include speed control of the motor (ie the distance to be covered by the rod) and a delay for which we expect the motor to wait for coating to occur.
We have tried to implement best user interface by adding lcd screen, matrix keyboard, web application for remote access, and pre-defined speed settings in our project. Since we want to implement this project in the new semiconductor fabrication laboratory of PEC, we want to use metallic or ceramic apparatus which can be used in a clean room.
A lot of effort has been paid to reduce vibrations and make the device portable. The unit can be run over Wifi via the website and the experiment logs are also maintained for future results. Thus, being a cheap alternative to market available dip coating units, our model is best suited for college laboratories and small industries to conduct fabrication and research.
